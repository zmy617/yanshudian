# 花冠专卖店永盛烟酒 - 手机端商品展示页

一个简单易用的烟酒店商品展示与管理系统，适合朋友圈分享和店铺扫码展示。

## 功能特点

### 顾客端 (index.html)
- 📱 **商品展示** - 支持分类筛选（香烟、白酒、礼盒）
- 🔍 **商品搜索** - 关键词实时搜索
- 🏷️ **规格选择** - 香烟（盒/条）、白酒（瓶/提/箱）、礼盒
- 🏷️ **商品标签** - 热销、新品标识
- 🛒 **购物车** - 加入购物车、数量调整、删除商品
- 📤 **微信分享** - 一键分享给朋友
- 📱 **一键拨打** - 点击电话直接拨打
- 📍 **地图导航** - 跳转地图应用
- ⭐ **店铺收藏** - 收藏店铺
- 📦 **二维码** - 生成店铺二维码供顾客扫码

### 管理后台 (admin.html)
- 📝 **商品管理** - 添加、编辑、删除商品
- 🖼️ **图片管理** - 支持粘贴图片链接
- 🏷️ **标签管理** - 设置热销、新品标签
- ⚙️ **店铺设置** - 修改店名、电话、地址
- 💾 **数据导出** - 导出商品数据备份
- 🔄 **数据重置** - 一键恢复默认商品

## 快速开始

### 方法一：本地打开（适合测试）

1. 双击 `index.html` 打开顾客端
2. 双击 `admin.html` 打开管理后台

### 方法二：部署到网上（推荐）

#### 部署到 GitHub Pages（免费）

1. 创建 GitHub 账号
2. 新建仓库，上传所有文件
3. 进入 Settings → Pages
4. Source 选择 "main branch"，保存
5. 等待几分钟后获取访问链接

#### 部署到 Vercel（免费，更快）

1. 注册 Vercel 账号
2. 导入 GitHub 仓库
3. 自动部署，获取链接

### 生成二维码

1. 访问你的网站链接
2. 点击底部「二维码」按钮
3. 长按保存二维码图片
4. 打印贴在店铺或发给顾客

## 使用指南

### 添加商品

1. 打开 `admin.html`
2. 点击「添加商品」标签
3. 填写商品名称、选择分类
4. 设置规格和价格（如：1盒=65元，1条=650元）
5. 可选：粘贴商品图片链接
6. 点击「保存商品」

### 修改商品

1. 打开 `admin.html`
2. 在商品列表点击「编辑」
3. 修改信息后保存

### 修改店铺信息

1. 打开 `admin.html`
2. 点击「店铺设置」
3. 修改店名、电话、地址
4. 点击「保存设置」

## 商品图片建议

### 获取白底商品图的方法

1. **自己拍照** - 用手机拍摄后，用 App 去背景（Remove.bg、图怪兽等）
2. **AI 生成** - 用 AI 工具生成商品示意图
3. **网络搜索** - 从网上保存合适的图片
4. **使用默认图** - 点击「使用默认占位图」按钮

### 推荐工具

- **Remove.bg** - AI 自动去背景
- **图怪兽** - 在线设计工具
- **Canva** - 设计商品图
- **稿定设计** - 电商素材模板

## 常见问题

### Q: 为什么图片显示不出来？
A: 检查图片链接是否正确，确保是可直接访问的 URL

### Q: 购物车数据会丢失吗？
A: 购物车数据保存在浏览器本地，更换设备后不会同步

### Q: 如何修改商品排序？
A: 在管理后台添加新商品时会放在最前面，编辑不影响顺序

### Q: 顾客能看到管理后台吗？
A: 不能，管理后台需要手动打开 `admin.html`
